Tem dúvida?
A gente responde.

O plano gratuito tem limite de tempo?+
Não. O plano Free é gratuito para sempre — não é um trial. Você pode usar os 10 monitores HTTP, a status page e os alertas por email sem pagar nada, pelo tempo que quiser.
Como funciona o alerta por WhatsApp?+
Você conecta seu número de WhatsApp nas configurações da conta. Quando qualquer monitor cair ou se recuperar, o UpStat manda uma mensagem direto pra você — sem app extra. Disponível no plano Pro.
O que é um Synthetic Monitor?+
É um monitor que simula ações reais de um usuário — sequências de requisições HTTP encadeadas, como login ou checkout completo. Detecta quando uma funcionalidade quebra, mesmo que o servidor continue de pé.
Como instalo e uso o SDK?+
Instale via npm: npm install upstat-sdk. Inicialize com sua API key (disponível no dashboard) e use os métodos para criar monitores, checar status e enviar heartbeats. Docs em upstat.online/docs.
Como meus usuários se inscrevem na status page?+
Na sua status page pública há um botão "Receber notificações". O usuário informa email ou WhatsApp e passa a receber alertas automáticos em cada incidente. Você vê a lista de inscritos no dashboard.
Posso usar a CLI no CI/CD?+
Sim. Basta rodar upstat monitors pause <id> antes do deploy e upstat monitors resume <id> depois para evitar falsos positivos. Funciona em qualquer pipeline (GitHub Actions, GitLab CI, etc.).
Qual a diferença entre monitorar cron jobs e APIs?+
Monitorar APIs é ativo: o UpStat faz requisições periódicas. Monitorar cron jobs é passivo (heartbeat): o seu job envia um ping pro UpStat após executar. Se o ping não chegar no intervalo esperado, o alerta dispara — ideal para backups, envio de emails e relatórios.
Preciso de cartão de crédito pra criar conta?+
Não. Conta grátis sem nenhum dado de pagamento. Cartão só é necessário para fazer upgrade pro plano Pro.
